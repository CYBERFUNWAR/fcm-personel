🔥 Connexion Firebase (partage multi-profs)

Pourquoi Firebase ?
Firebase permet de partager les fiches entre plusieurs professeurs sur n'importe quel appareil, en temps réel.

📌 C'est gratuit pour votre usage (jusqu'à 50 000 lectures/jour).
📌 Vos données restent privées dans votre propre projet Firebase.
📌 console.firebase.google.com → Créer un projet → Ajouter une app Web → Copier la config ci-dessous.
Étapes rapides :
  1. Aller sur console.firebase.google.com
  2. Créer un projet (ex : "FCM-Full-Contact")
  3. Dans le projet → Firestore Database → Créer une base de données → Mode test
  4. Dans Paramètres du projet → Ajouter une application Web
  5. Copier les valeurs du fichier de config et les coller ci-dessous
  6. Partager ce fichier HTML avec vos collègues — ils verront les mêmes élèves !
